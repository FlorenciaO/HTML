Hyperlink tag <a>

The tag is an <a> because it comes from anchor, just like its behavior. If you click on it, it will take you to the specified URL or to a section on the document as well. The href attribute is where you put the link so it can work as expected. In case you want to link a section on the document, you must add # followed by the id of that section.
Another important attribute is the target. If you want to open the link on another tab, fill the attribute with "_blank".
